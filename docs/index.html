C2 Log-Parser

Streamline your red team reporting.

Parses C2 log files, stores them in a SQLite database, and generates comprehensive CSV reports. Built for Cobalt Strike, Brute Ratel, and Outflank C2 (OC2).

View on GitHub

Ingest

Parse log files seamlessly and store them securely in a structured SQLite database.

Minimize

Remove clutter automatically. Exclude specific beacons using a simple configuration file.

Report

Generate detailed CSV reports ready for delivery, including TIBER-EU formats.

Quick Start

# Setup environment
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt

# Full pipeline: ingest, minimize, report
python3 gimmelogs.py -x cs -i ./logs -c config.yml -m -r -p ./results
